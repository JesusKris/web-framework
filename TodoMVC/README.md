# Mist.js - TodoMVC

## **Usage**
Navigate to todomvc
```bash
cd ./todomvc
```
Install node modules
```bash
npm install
```
Run web-bundler
```
npm run dev
```

## **Documenation**
Run ./docs/index.html with [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) VSCode extension.
